The taxa trainers are fasta files containing the sequences of known species or genera and are used to assign the taxonomy to the sequences provided by your Illumina Analysis.
With the App come the following taxa trainer files:
hitdb - the human intestinal data base

this is a taxonomy reference of human intestinal 16S rRNA genes that were described in
Ritari et al. BMC Genomics (2015) 16:1056
DOI 10.1186/s12864-015-2265-y

rdp - the ribosomal database project

this is a training set created from data of the ribosomal database project
Cole et al. Nucl. Acids Res. (2014)
DOI:10.1093/nar/gkt1244

silva - the SILVA project

this is a training set created from data of the SILVA project
Yilmaz et al. Nucl. Acids Res. (2014)
DOI:10.1093/nar/gkt1209
